More Paths

if / else if / else

if (score >= 90) { // A } else if (score >= 80) { // B } else { // C or below }

Checks in order, top to bottom. First true one wins.

⚠ Gotcha

Order Matters

Bug: everyone gets an A?!

if (score >= 60) ... else if (score >= 90) ...

Fix: check highest first

if (score >= 90) ... else if (score >= 60) ...

If we check >= 60 first, a 95 matches immediately and never reaches the A check.

Randomness

Math.random()

Math.random() → 0.4817...

Returns a random decimal from 0 up to (but not including) 1.

Useful, but we want a whole number.

Whole Numbers

Math.floor()

Math.floor(4.9) → 4 // rounds down
Math.floor(Math.random() * 100) + 1 → 1–100

Multiply the random decimal by 100, floor it, add 1 — that's our secret number.
